# 🛒 Full Stack E-Commerce Website (MERN Stack)

A complete **Full Stack E-Commerce Website** built with the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**. This project includes a modern user interface, secure authentication, shopping cart functionality, order management, online payment integration, and an admin dashboard for store management.

## 🚀 Live Demo

🌐 **Website:** https://ecommerce-app-frontend-topaz.vercel.app

## 📖 Project Overview

The application allows users to:

- Browse and explore products
- Search, filter, and sort products
- Select product variants (e.g., size)
- Add products to the shopping cart
- Place orders with delivery address details
- Pay using Cash on Delivery (COD) or Online Payment
- Track and manage orders

The project also includes an **Admin Dashboard** where administrators can:

- Add new products
- Upload product images
- Delete products
- View all products
- Manage customer orders

All product, user, and order data are stored securely in a **MongoDB database**. The application is deployed online using **Vercel**.
